# Introduction to Wolfgang D20 Dice

Wolfgang D20 Dice is a simple yet powerful library for simulating dice rolls in .NET applications.

## Why Use This Library?

- **Simple API**: Easy to understand and use
- **Cross-platform**: Works on .NET Framework, .NET Standard, and modern .NET
- **Reliable**: Uses proven random number generation techniques
- **Lightweight**: Minimal dependencies and small footprint

## Use Cases

- Game development
- Tabletop RPG simulators
- Probability demonstrations
- Random number generation for any application
