Remarks

The average roll is the midpoint of the possible results, (MinValue + MaxValue) / 2, which is always a whole number or a half. These methods round that midpoint to an int, which is convenient for tabletop rules that call for average damage rounded a particular way — for example, half damage on a successful save is conventionally rounded down.
